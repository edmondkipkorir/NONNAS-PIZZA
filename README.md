### NANA'S -PIZZA-DELICACY
This is a web application where customers order as many pizzas as they want with different sizes and toppings.All you have to do is make an order of any pizza of your choice and it will be delivered almost immidiately to your doorstep
## Technology used.
html
css
javascript
Bootstrap
## Prerequisites
install vs code
## Setup/Installation
Open your terminal (ctrl+alt+T).
Initialize git on your terminal.
git clone https://github.com/edmondkipkorir/NONNAS-PIZZA.
open the 'NANA'S-PIZZA-DELICASY' folder.
Open the file named index.html on your preferred browser.
Enter your pizza of choice into the input boxes.
Click Submit.
